# Шаблон: Остатки и обороты

## Назначение
Использовать для отчетов по остаткам, приходу, расходу, оборотам и складским движениям.

## Что уже входит
- геро-заголовок и корпоративный layout
- KPI-карточки:
  - позиции
  - на начало
  - приход
  - расход
  - на конец
- основная таблица с внутренней прокруткой
- правая панель фильтров
- фильтры через справочники `catalog.*`
- паттерн `title -> id` для HubCloud
- mock mode + HubCloud mode

## Когда брать этот шаблон
- если в отчете есть склад
- если есть период
- если есть обороты по номенклатуре
- если нужны выпадающие фильтры, показывающие `title`, но передающие `id`

## Следующие шаги после scaffold
1. Обновить `REPORT_SPEC.md`
2. Заменить `DS.txt`
3. Адаптировать `buildHubCloudExpression()`
4. Проверить `mock-data.json`
5. Прогнать:
   - `node ..\\tools\\hc-encoding-gate.js`
   - `node ..\\tools\\hc-factory-lint.js reports/<report-name>`
